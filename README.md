# Markdown Publisher

[中文说明](README.zh-CN.md)

Markdown Publisher is a Chrome extension for publishing one Markdown article to multiple platforms from a single workflow.

## Overview

Markdown Publisher helps you publish the same article faster without repeating the same manual steps on every platform.

Supported channels:

- X
- dev.to
- Reddit
- Medium
- Substack

## What it is for

- Cross-posting one article to multiple platforms
- Keeping titles and content more consistent
- Reducing repeated copy-and-paste work
- Speeding up publishing for creators, writers, and small teams

## How it works

1. Choose the channels you want to publish to.
2. Select a Markdown file or a folder that contains Markdown files.
3. Review the parsed title and tags.
4. Open the publishing pages for the selected platforms.
5. Continue publishing on each platform.

## GitHub Releases

Each GitHub Release includes a packaged Chrome extension ZIP file for download. It can be used for manual installation or later submission to the Chrome Web Store.
