Markdown Publisher

Write once, publish everywhere 一次编写，多平台发布

A Chrome extension that distributes your Markdown article to multiple publishing platforms in one click. 一个 Chrome 扩展，一键将 Markdown 文章分发到多个发布平台。

Download from GitHub 从 GitHub 下载

Features 功能特性

Everything you need to streamline your cross-platform publishing workflow. 简化跨平台发布流程所需的一切功能。

Multi-Platform Publishing 多平台发布

Publish to X, X Articles, Dev.to, Reddit, Medium, and Substack from a single interface. 通过一个界面即可发布到 X、X Articles、Dev.to、Reddit、Medium 和 Substack。

Markdown Support Markdown 支持

Write in Markdown and the extension handles format conversion for each platform automatically. 使用 Markdown 编写，扩展会自动为每个平台处理格式转换。

One-Click Fill 一键填充

Automatically fill in title, content, and tags on each platform's editor with a single click. 一键自动在每个平台的编辑器中填入标题、内容和标签。

Tags Management 标签管理

Define tags once and have them applied consistently across all supported platforms. 只需定义一次标签，即可在所有支持的平台上统一应用。

Clipboard Image Paste 剪贴板图片粘贴

Paste images directly from your clipboard into the article content for quick embedding. 直接从剪贴板粘贴图片到文章内容中，快速嵌入。

Open Source 开源项目

Fully open source. Inspect the code, contribute improvements, or customize it to your needs. 完全开源。可以查看代码、贡献改进或根据需求自定义。

How It Works 使用流程

Four simple steps from draft to published across all platforms. 从草稿到多平台发布，只需四个简单步骤。

1

Write in Markdown 编写 Markdown

Compose your article in your favorite Markdown editor. 在你喜欢的 Markdown 编辑器中撰写文章。

2

Paste into Extension 粘贴到扩展

Open the extension popup and paste your Markdown content, title, and tags. 打开扩展弹窗，粘贴 Markdown 内容、标题和标签。

3

Select Platforms 选择平台

Navigate to each target platform and click the fill button. 导航到各目标平台，点击填充按钮。

4

Publish 发布

Review and hit publish on each platform. Done! 在每个平台上检查并点击发布，完成！

Get Started 开始使用

Install the extension in under a minute. 不到一分钟即可安装扩展。

Installation Steps 安装步骤

  1. Download the latest release .zip file from the GitHub Releases page. 从 GitHub Releases 页面下载最新的 .zip 文件。
  2. Unzip the file to a folder on your computer. 将文件解压到电脑上的一个文件夹。
  3. Open Chrome and navigate to chrome://extensions. 打开 Chrome 并导航到 chrome://extensions。
  4. Enable Developer mode in the top-right corner. 在右上角启用开发者模式。
  5. Click Load unpacked and select the unzipped folder. 点击加载已解压的扩展程序，选择解压后的文件夹。
